# Release v0.4.1-etapa4.1a — Optimización de Excel dinámico

## Estado general

ETAPA 4.1a aprobada técnicamente.

Se optimizaron las rutas que usaban Excel moviendo la librería xlsx desde imports estáticos a imports dinámicos dentro de los handlers de importar/exportar.

## Cambios principales

- Eliminados imports estáticos de xlsx.
- xlsx ahora se carga solo cuando el usuario usa funciones de Excel.
- Reducción importante del First Load JS en rutas con Excel.
- No se cambió lógica de negocio.
- No se cambió diseño.
- No se tocaron rutas.
- No se tocó el flujo crítico de consumo.
- No se agregaron dependencias.
