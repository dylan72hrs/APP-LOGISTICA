Se agrego `backend/php-api` como estructura PHP simple compatible con cPanel tradicional.

Incluye:

- Front controller minimo.
- Router simple.
- Helper de respuestas JSON.
- Controlador de salud.
- Stub de conexion a base de datos.
- Stub de middleware de auth.
- Configuracion de base de datos de ejemplo sin credenciales reales.
