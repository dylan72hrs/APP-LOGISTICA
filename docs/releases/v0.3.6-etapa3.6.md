# Release v0.3.6-etapa3.6 — Hotfix estabilidad, perfil visual y login

## Estado general

ETAPA 3.6 aprobada con observaciones.

Se corrigió el error crítico de Genkit/Gemini cuando no existe GEMINI_API_KEY o GOOGLE_API_KEY, se agregó perfil visual básico y se mejoró la pantalla de login sin implementar Firebase, Firestore, Auth real, roles reales ni permisos reales.

## Cambios principales

- Corrección del crash de Genkit por falta de API key.
- /restock ya no rompe la app si no hay GEMINI_API_KEY o GOOGLE_API_KEY.
- /restock muestra estado claro de IA no configurada.
- Botón de generar sugerencias queda deshabilitado cuando no hay API key.
- Perfil visual funcional desde el menú superior.
- Perfil muestra foto, nombre, correo, rol visual y apodo editable.
- Apodo guardado localmente.
- Login mejorado visualmente con robot decorativo CSS/Tailwind.
- Formulario de login actual se mantiene.
