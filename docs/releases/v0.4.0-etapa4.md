# Release v0.4.0-etapa4 — Flujo crítico de consumo

## Estado general

ETAPA 4 aprobada técnicamente.

Se reforzó el flujo principal de consumo para evitar registros incompletos, inconsistentes o con stock insuficiente.

## Cambios principales

- Validación obligatoria de bodega específica.
- Validación de trabajador seleccionado y existente.
- Validación de proyecto seleccionado y existente.
- Validación de productos agregados.
- Validación de producto existente en inventario.
- Validación de producto perteneciente a la bodega activa.
- Validación de stock disponible.
- Validación de cantidad numérica, mayor que cero y no superior al stock.
- Validación de costo unitario válido.
- Validación de total general válido.
- Revalidación antes de enviar a espera.
- Revalidación antes de registrar consumo.
- Prevención de descuentos parciales.
- Limpieza de productos/vales pendientes al cambiar de bodega.
