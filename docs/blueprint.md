# **App Name**: StockFlow

## Core Features:

- User Authentication: Secure user login and registration with role-based access control (admin, operator, reports).
- Warehouse Management: Create and manage multiple warehouses (bodegas) with individual inventories.
- Inventory Management: Add, update, and track EPP inventory. Support single item addition and bulk upload via Excel (code, description, size, quantity, cost). Automatically update existing items or create new ones.
- Consumption Tracking: Record EPP consumption by worker and project. Verify stock availability. Print consumption vouchers for worker signature.
- Reporting and Analytics: Generate reports filtered by project, worker, and date range. Export data to Excel. Create reports that summarize trends.
- Master Data Management: Manage worker and project master data, if preloaded.
- Intelligent Restock Suggestions: AI powered tool to give recommendations for stock replenishment.

## Style Guidelines:

- Primary color: Navy blue (#2E3192) for trust and professionalism.
- Background color: Light gray (#EAEAF2), offering a clean, neutral backdrop.
- Accent color: Sky blue (#72BCD4) to highlight key actions and elements.
- Body and headline font: 'Inter', a grotesque sans-serif providing a modern, machined, objective look.
- Use clear and consistent icons for navigation and actions.
- A clean and well-structured layout with clear information hierarchy. Use of cards and tables to display data effectively.
- Subtle transitions and animations for loading states and user interactions. Smooth scrolling and feedback animations for button clicks.
